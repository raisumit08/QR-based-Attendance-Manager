About Us

The web application allows users to take attendance of no. of people easily with QR. Users can take attendance and can see realtime data. The web application also provides real-time data synchronization, ensuring that the data is always up to date.

The purpose of this is to document the development of a Progressive web application using HTML, CSS, JavaScript, Firebase as the backend database, and APIs. The web application is designed to be a platform for users to store, retrieve and manage their data. The project aims to provide a secure and efficient solution for users to store and manage their data online.
